# Sistema Inteligente de Monitoramento de Ambiente
### Desafio 2 — Sistemas Embarcados e IoT
**Disciplina:** Engenharia de Computação — UNICET
**Período:** 6º Período

---

## Descrição do Projeto

Sistema de simulação de sensores IoT que monitora temperatura,
umidade e luminosidade de um ambiente em tempo real.
O sistema detecta situações críticas automaticamente e gera
um relatório final com as estatísticas das leituras.

---

## Funcionalidades

- Simulação de 3 sensores: Temperatura, Umidade e Luminosidade
- Geração automática de 50 leituras por execução
- Sistema de alertas automáticos para valores fora da faixa segura
- Dashboard no terminal atualizado a cada leitura
- Histórico completo de leituras com timestamp
- Relatório final salvo em `relatorio_monitoramento.txt`

---

## Tecnologias Utilizadas

- Python 3.8+
- Bibliotecas: `random`, `time`, `datetime` (nativas)
- Paradigma: Programação Orientada a Objetos (POO)
